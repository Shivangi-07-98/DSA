https://leetcode.com/problems/count-complete-tree-nodes/description/

time O((log n)^2)
space O(log n)

class Solution {
  public int countNodes(TreeNode root) {
    if (root == null) {
      return 0;
    }

    int lh = 0;
    TreeNode node = root.left;
    while (node != null) {
      node = node.left;
      lh++;
    }

    int rh = 0;
    node = root.right;
    while (node != null) {
      node = node.right;
      rh++;
    }

    if (lh == rh) {
      int ht = lh + 1;
      return (1 << ht) - 1;
    } else {
      return countNodes(root.left) + countNodes(root.right) + 1;
    }

  }
}

complete binary tree formula, where h is height in terms of node 2^(h+1) - 1
complete binary tree mai saare level pure honge
aur nhi honge toh 3 level hai toh, 2 level pura hoga aur 3rd level left sae pura hoga
