recursion - time O(n), space height (max stack mai ht ke barabar hi bnde ho skte hai)

morris - time O(n), space O(1) iteration, does not take space

class Solution {
  public List<Integer> inorderTraversal(TreeNode root) {
    List<Integer> retVal = new ArrayList<>();

    TreeNode curr = root;

    // ekdum end mai left right null hota hai, mtlb nodes khatam hogye, toh iteration stop krne ke liye
    while (curr != null) {
      if (curr.left == null) {
        retVal.add(curr.val);
        curr = curr.right;
      } else {
        TreeNode iop = curr.left;
        while (iop.right != null && iop.right != curr) {
          iop = iop.right;
        }

        if (iop.right == null) {
          iop.right = curr; // making the link
          curr = curr.left;
        } else {
          iop.right = null;
          retVal.add(curr.val);
          curr = curr.right;
        }
      }
    }

    return retVal;
  }
}


Morris Traversal (Inorder) - Node

Case 1: node.left == null
print node
go to right

Case 2: node.left != null
left not processed (left ka rightmost)
iop.right == null
iop.right = node (return path)
go to left

Case 3: node.left != null
left is processed (left ka rightmost)
iop.right == node
iop.right = null
print node
go to right
